PCA for Anomaly Detection

June 1, 2021

Collaborators: Tim Wickard

Description:

  • This project was done as consulting work for a manufacturing plant which had a vast amount of process data from their plant’s sensor network (data have been anonymized here).
  • The goal was to build a model to detect anomalies in the data that could contribute to defects in the manufactured extruded multi-ply plastic sheet material.
  • We used Principal Component Analysis (PCA) to find anomalous data and learn which features in the sensor network contributed the most to this anomalous behavior.
  • Additionally, we performed Sparse PCA (to help learn the significant features) and Kernel PCA (to account for non-linear separation of data into ’normal’ versus ‘anomalous’).
  • The algorithm was written in both Python and JavaScript.
  • The GitHub repository will also feature a tutorial on modeling and implementing anomaly detection on general multivariate process data, including intuitive and mathematical explanations.
