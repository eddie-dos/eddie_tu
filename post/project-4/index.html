Project RattleCam - processing raw data

November 27, 2022

Collaborators: Dr. Scott Boback (Dickinson College), Dr. Emily Taylor (Cal Poly)

General Project Description: Project RattleCam is a community science project created to help study and to educate the public about Prairie Rattlesnakes. Educating the public on these often maligned creatures is vital work towards the conservation of snakes, which play a crucial role in the balance of their ecosystems. The project has curated more than 60,000 photos of these snakes using remotely-deployed cameras. Using the online community science platform Zooniverse, we have enlisted thousands of citizen scientists who provided useful annotations of photos by addressing various questions (e.g. number of snakes, snake behavior), giving us valuable data rich for analysis.

Sub-project Description:

  • Each photo has a collection of Zooniverse users who annotated the photo
  • Our Goal: clean and aggregate the user-response data for analysis
  • Algorithms were built to create counts of different categorical responses by users
  • Text mining was needed to clean and extract numerical values from the “count” responses, since inputs on Zooniverse were given as free text responses.
  • Various methods of aggregating user responses to create labels for each photo
    • using an agreement index (in categorical responses) to determine appropriate labels for photo
    • using voting methods with varying cut-off values to determine appropriate labels
    • using mean/median to give label for numerical responses
